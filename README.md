# Machine Learning Programming
**EPFL – Machine Learning Programming - MICRO-401**

## Overview

This project was developed as part of MICRO-401 – Machine Learning Programming, a practice-based course at EPFL focused on implementing, testing, and analyzing core machine learning algorithms from scratch.
The goal is to deepen understanding of machine learning concepts by building algorithms and rigorously evaluating them on real-world datasets.

## Objectives

- Implement classical machine learning algorithms  
- Perform performance evaluation using standard metrics  
- Apply cross-validation and hyperparameter search techniques  
- Analyze results with ROC curves, F-measure, and other relevant criteria

The project emphasizes both correct implementation and interpretation of results.

## Algorithms Implemented

Typical algorithms developed in this project include:

- k-Nearest Neighbors (k-NN)
- Linear and logistic regression
- Support Vector Machines (SVM)
- Decision trees / Random forests
- Clustering methods (e.g., k-means)
- Dimensionality reduction techniques (PCA, etc.)
- Neural Network (NN)
- Gaussian Mixture Model (GMM)

## Performance Evaluation

Evaluation techniques applied:

- Cross-validation for robust model assessment  
- Grid search / hyperparameter tuning  
- Confusion matrix, F-measure, ROC curve, AUC

These evaluation methods ensure that models are not just coded correctly, but also perform well in practice.

## Usage

1. Load dataset (CSV / standard format)  
2. Preprocess data (normalization / feature extraction)  
3. Train model  
4. Evaluate with cross-validation  
5. Plot and analyze results
